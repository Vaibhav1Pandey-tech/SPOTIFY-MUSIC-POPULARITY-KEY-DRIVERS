Overview

This project analyzes what drives music popularity on Spotify using data visualization.
The dashboard highlights how artist reach, genres, release type, and explicit content impact track popularity.

Objective

To identify the key factors influencing Spotify track popularity and present them through a business-focused dashboard.

Dataset

Source: Spotify dataset (Kaggle)

~9,000 tracks with popularity, genre, artist reach, release type, and explicit flag


Key Insights

Track popularity increases with artist reach

Pop & Hip-Hop genres perform best

Singles outperform albums on average

Explicit content has no guaranteed impact on popularity
